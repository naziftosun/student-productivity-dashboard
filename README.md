# Student Productivity Dashboard

A modern web application that helps students manage their academic life — courses, assignments, deadlines, weekly planning, and progress tracking — all in one place.

> Built as a portfolio project to demonstrate real-world React, TypeScript, and state management skills.

---

## Features

- **Course Manager** — Add and organize courses with color labels
- **Assignment Tracker** — Create tasks linked to courses with due dates and priority levels
- **Deadline Calendar** — Visual week/month view of upcoming deadlines
- **Weekly Planner** — Plan your week with a drag-and-drop daily board
- **Progress Dashboard** — Track completion rates, streaks, and upcoming workload

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 18 + TypeScript |
| Build Tool | Vite |
| Styling | Tailwind CSS + shadcn/ui |
| State | Zustand (persisted to localStorage) |
| Routing | React Router v6 |
| Deployment | Vercel |

---

## Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/student-productivity-dashboard.git
cd student-productivity-dashboard

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
